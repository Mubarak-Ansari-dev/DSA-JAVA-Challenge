342. Powe of Four
Solution:-

      class Solution {
    public boolean isPowerOfFour(double n) {
        if(n==1)
         return true;
        else if(n<1)
         return false;
         else
          return isPowerOfFour(n/4.0); 
        
    }
}
        
Example 1:

Input: n = 16
Output: true
Example 2:

Input: n = 5
Output: false
Example 3:

Input: n = 1
Output: true
